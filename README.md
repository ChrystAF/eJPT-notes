# eJPTv2 Study Notes 

While pursuing the eLearnSecurity Junior Penetration Tester (eJPT) certification, my goal was to validate my practical cybersecurity skills and demonstrate proficiency in real-world penetration testing techniques. The exam required completing a hands-on assessment within a 48-hour window, where I needed to perform reconnaissance, enumeration, exploitation, and lateral movement across a simulated network to correctly answer all exam questions. I systematically approached the environment by mapping the network using Nmap, identifying vulnerabilities through service enumeration, and exploiting weaknesses using tools such as Metasploit. After gaining initial access, I leveraged reverse shells and network pivoting techniques to explore additional systems. Throughout the process, I documented commands, findings, and methodologies to maintain a clear record of the engagement. By successfully compromising the target systems and answering all exam questions, I earned the eJPT certification and demonstrated the ability to apply foundational penetration testing methodologies in a realistic, simulated environment.
